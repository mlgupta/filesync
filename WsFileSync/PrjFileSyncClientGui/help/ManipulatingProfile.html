Manipulating Existing Profile  

Editing a profile: (To edit an existing profile)

Select desired profile from the list of profiles at 'Sync Profile' window.

Click on the 'Edit' icon [ ] which allows you to edit the profile name and copy options for synchronization. User is able to edit the local and remote folder path only once, but can edit the profile name and copy options many a times.

Deleting a profile: (To delete an existing profile)

Select a desired profile to be deleted.


Progress bar appears to show progress while downloading. 

Click on the 'Delete' icon [ ].

Click 'Yes' to confirm delete or 'No' to abort deleting.
