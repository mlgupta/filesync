Viewing properties of Documents/Folders  

How to see the properties of Document/Folder?

In Folder/Document List select the folder/document for which you would like to see the Property.

Click on the Property icon [ ] or select from menu Remote->Properties to view properties of remote item(s) or Click on the Property icon [ ] or select from menu Remote->Properties to view properties of remote item(s) or Local->Properties to view properties of local item(s)

You can rename the folder/document through property page.

Press 'OK' to save the changes made.

Press 'Cancel' to abort 'Rename'.
