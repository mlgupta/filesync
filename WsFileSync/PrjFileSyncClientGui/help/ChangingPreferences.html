Changing Preferences  

JXTA preferences:

Changing JXTA preferences includes changing Socket address,Rendezvous & relay address.

How to change Jxta preferences?

Select File->Preferences menu option at FileSync startup window resulting in 'Jxta Preferences' window pop-up with default Socket, Rendezvous & relay address.

 

Enter a valid Socket Address that matches with server 

Change Rendezvous & relay address to the desired value.

Click 'Ok' to save these changes.

Click 'Cancel' to abort saving.

Application needs to be restarted to take effect these changes.
