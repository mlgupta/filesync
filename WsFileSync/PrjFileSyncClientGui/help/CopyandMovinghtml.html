Copying or moving Documents/Folders  

How to copy Folder(s)/Document(s) to other folder?

In Folder/Document List select the folder/document you would like to copy.

You can select multiple folder/document.

Click on the 'Copy' icon [ ] or select from menu Remote->Copy to copy from remote browser or Local->Copy to copy from local browser. 

Navigate to the folder where you would like to Copy the selected items.

Click on the 'Paste' icon [ ] or select from menu Remote->Copy to paste into remote browser or Local->Paste to paste into local browser. 


Progress bar appears to show progress while copying.

Press 'Cancel' to abort copying.

Press 'Detail…' to view the details of copy such as Total Size, Data Transferred, Data Remaining, Total time, Time Elapsed, Time Remaining, speed. 

How to move Folder(s)/Document(s) to other folder?

In Folder/Document List select the folder/document you would like to move. You can select multiple folder/document.

Click on the 'Cut' icon [ ] or select from menu Remote->Cut to move from remote browser or Local->Copy to move from local browser. 

Navigate to the folder where you would like to move the selected items.

Click on the 'Paste' icon [ ] or select from menu Remote->Copy to paste into remote browser or Local->Paste to paste into local browser. 

Progress bar appears to show progress while moving. 

Press 'Cancel' to abort moving.
