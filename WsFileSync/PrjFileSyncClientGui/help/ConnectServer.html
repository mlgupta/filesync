Connecting  Remote Server  

To be able to view and manipulate the server filesystem on its local counterpart, user need to connect with remote server with a userid and password provided by administrator.

Connecting with server:

Select File->Preferences menu option at FileSync startup window resulting in 'Jxta Preferences' window pop-up with default Socket, Rendezvous & relay address.

 

Enter the userid and password ,select the 'Remember my ID on this computer' option if you want the computer should remember your userid and password. So that when you login next time it automatically sets them at the login prompt.

Click 'Ok' to connect with server.

Click 'Cancel' to abort connecting server.

After validating the userid and password a view of remote filesystem appears in the remote browser which could be manipulated by the user.
