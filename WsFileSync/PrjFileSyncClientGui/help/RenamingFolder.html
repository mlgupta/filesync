Renaming a Document/Folder  

How to rename a Document/Folder?

In Folder/Document List select the folder/document to be renamed.

Click on the 'Rename' icon [   ] or select from menu Remote->Rename to rename a remote folder Local->Rename to rename a local folder.

Old name appears for the selected Document/ Folder.

Enter the name to which you would like to rename.

Press 'Ok' to bring into effect the name you have changed.

Press 'Cancel' to abort renaming.

Unable to rename a Document/Folder

Insufficient rights to rename a folder in the selected location
