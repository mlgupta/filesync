Navigating through Folders  
User can move back and forth in the navigation history i.e. through the folders which he navigated earlier. Moving back in the navigation history.

Click the [ ] button from standard toolbar, which takes you one step back in the navigation history. Accordingly the navigation panel at left and Folder/Document List panel at right gets refreshed. 

Moving Forward in the navigation history

Click the [  ] button from standard toolbar, which takes you one step back in the navigation history. Accordingly the navigation panel at left and Folder/Document List panel at right gets refreshed. 

Moving up to the parent folder

Click the [   ]button from application window toolbar,toolbar, which takes you to the parent folder of current folder. Accordingly the navigation panel at left and Folder/Document List panel at right gets refreshed. 

Refreshing browser

Click the [  ]button from application window toolbar, which which gives a refreshed view of the contents of current folder. Accordingly the navigation panel at left and Folder/Document List panel at right gets refreshed.
