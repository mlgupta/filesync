Uploading Documents/Folders to the Remote Server  

How to upload Folder(s)/Documents?

In Folder/Document List of local browser select the folder/document you would like to upload.

You can select multiple folders/documents.

Navigate to the folder in the remote browser where you would like to upload the selected items.

Click on the 'Upload' icon [ ] or select from menu Local->Upload to start Uploading.


Progress bar appears to show progress while uploading. 

Press 'Cancel' to abort uploading.

Press 'Detail…' to view the details of upload such as Total Size, Data Transferred, Data Remaining, Total time, TimePress 'Detail…' to view the details of upload such as Total Size, Data Transferred, Data Remaining, Total time, Time Elapsed, Time Remaining, speed.


After the completion of upload remote browser is refreshed to view the uploaded contents.

Unable to upload.

Broken connection with remote server.

Insufficient access to create new folders/documents in navigated remote folder.
